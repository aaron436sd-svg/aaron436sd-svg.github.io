Admin — Update Today's Offer

This simple editor stores the daily offer in your browser (localStorage). Open this page on the restaurant manager's computer, update the fields and click Save. The website will read and show the saved offer on the homepage.

Note: this is a lightweight editor for the restaurant owner to update the visible daily menu. For a server-backed CMS, ask me to add a simple Node/Netlify function integration.
